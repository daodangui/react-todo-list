## My Todo List Manager

基于 React 的 todo-list App

### 社区技术依赖

- moment v2.19.1
- react-beautiful-dnd v10.0.0

### 组件封装

#### Checkbox

##### 说明：

多选框

##### 属性：

| 属性    | 类型    | 可选值 | 默认值 | 说明         |
| ------- | ------- | ------ | ------ | ------------ |
| checked | Boolean | -      | false  | 当前选中状态 |

##### 事件：

| 事件名称 | 回调参数 | 说明             |
| -------- | -------- | ---------------- |
| onChange | event    | 当改变状态时回调 |

#### Button

##### 说明：

##### 事件：
